# StudyBuddy.ai — AI-Powered Learning Assistant

StudyBuddy.ai is a modern AI-powered learning assistant that understands your syllabus, analyzes previous year question papers, answers study-related queries, summarizes videos, and even solves math problems.
Built with FastAPI, modular agents, Chroma vector search, HuggingFace embeddings, and Groq LLM — this project brings GenAI into education with real-world engineering practices.
